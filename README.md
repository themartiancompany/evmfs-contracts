# Ethereum Virtual Machine File System (EVMFS) Contracts

Solidity source code contracts of the Ethereum Virtual Machine File System (EVMFS).

This repository is a submodule for the
[main](
  https://github.com/themartiancompany/evmfs) one.

### Contracts

### Version 1.0

Version 1.0 is composed by the following contracts.

- [FileSystem](
    FileSystem.sol),
- [LengthLock](
    LengthLock.sol),
- [CrossChainFileSystem](
    CrossChainFileSystem.sol)

### Documentation

Documentation is available in
the `docs` directory in the main
repository.

The submodule name for the documentation
is `evmfs-docs`.
